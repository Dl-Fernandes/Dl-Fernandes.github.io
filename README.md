# 🌐 Currículo Web — Daniel Lima Fernandes

Este repositório contém o código-fonte do meu currículo profissional formatado como uma página web estática, hospedado gratuitamente através do **GitHub Pages**.

## 🚀 Acesso Direto
O resultado final pode ser visualizado em tempo real pelo link:
👉 **https://Dl-Fernandes.github.io**

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica dos dados profissionais.
* **CSS3:** Estilização responsiva (otimizada para computadores e dispositivos móveis) utilizando variáveis nativas e arquitetura limpa.
* **GitHub Pages:** Hospedagem e deploy automatizado da aplicação.

## 🎯 Objetivo do Projeto
Demonstrar na prática o uso do ecossistema Git/GitHub para a publicação, versionamento e gerenciamento de ambientes web estáveis, alinhado com meus estudos em Infraestrutura de TI e Suporte Técnico.
